# Spacify

## Description

Spacify is an easy, short and fun project that returns the "spacified" version of some text passed as argument to the script.

## How to use

- Requires Python 3.8+
- `<text>` is a required argument.

```sh
git clone https://github.com/qexat/spacify; cd src
python spacify <text>
```
